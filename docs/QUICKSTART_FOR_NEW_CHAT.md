> - Zero-cost infrastructure mandate
> - Modular architecture

## Tips for the New Chat

1. **Be Specific**: Reference the architecture documents frequently
2. **Test Early**: Ask the AI to test features as they're built
3. **Iterate**: Don't hesitate to request changes or improvements
4. **Validate**: Check that foundational principles are followed
5. **Document**: Ask for documentation as features are completed

## Troubleshooting

If the AI gets stuck:
- Reference the specific document (e.g., "Check DATABASE_SCHEMA.md for the table structure")
- Ask for a checkpoint and review progress
- Break down complex features into smaller steps
- Request specific examples or test cases

## Contact & Support

If you need help or have questions:

- Refer to foundational principles for design decisions
